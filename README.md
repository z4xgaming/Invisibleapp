# 👻 Invisible Name Store App

A powerful web app to generate and unlock **unique invisible & stylish names** using a coin-based system.

---

## 🚀 Features

- 💰 Coin System (Low / Medium / Premium)
- 🗂️ Separate Name Database (JSON files)
- 🔄 Auto Update (No Firebase required)
- 📋 One-click Copy
- 🎮 Gaming Friendly (Free Fire / BGMI)
- 📱 Easy APK Conversion (Web2App)

## ⚙️ How To Use

1. Open the project folder  
2. Run `index.html` in browser  
3. Select category:
   - Low Coins 💸
   - Medium Coins 💰
   - Premium Coins 💎  
4. Click **Unlock**  
5. Name will be copied automatically ✅

1. Open Web2App

3. Fill app details  
4. Click **Build App**  
5. Download APK 🎉

## 🔄 Auto Update System

- App loads data from JSON files  
- Updates automatically every 10 seconds  
- No Firebase or backend required
